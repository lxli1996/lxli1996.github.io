Ubuntu 安装中文输入法

2020年12月5日 14:03
，归类于 解决方案

安装fcitx，Ubuntu上一般自带

sudo apt-get install fcitx

安装谷歌拼音输入法

sudo apt-get install fcitx-googlepinyin

启用fcitx

进入系统设置(System Setting)->语言支持(Language Support)

在最后一项即keyboard input method system中选择fcitx。

重启电脑

在终端输入

fcitx-configtool

进入设置页面

点击左下角的+号添加输入法，在only show curent language这里不要选中，如果你的系统是英文的，就不会显示安装的中文输入法。

选择Google Pinyin，再点击OK。
