Ubuntu 1604 Ceres安装

2020年12月5日 14:03
，归类于 解决方案
  1. 首先安装eigen3，版本为3.3.7。下载地址：https://gitlab.com/libeigen/eigen/-/releases

安装：

mkdir build
cd build
cmake ..
sudo make install
  1. 安装ceres相关依赖
sudo apt-get install liblapack-dev
sudo apt-get install libsuitesparse-dev
sudo apt-get install libcxsparse3.1.2
sudo apt-get install libgflags-dev
sudo apt-get install libgoogle-glog-dev
sudo apt-get install libgtest-dev

如果安装时找不到 cxsparse 或者其他的lib，需要添加源

首先打开源的列表：sudo gedit /etc/apt/sources.list

把下面的源粘贴到source.list的最上方

deb http://cz.archive.ubuntu.com/ubuntu trusty main universe

更新一下：sudo apt-get update

  1. 编译ceres

首先下载ceres-1.13.0版本，地址：ceres-solver.org/ceres-solver-1.13.0.tar.gz

然后就是cmake、make、make install三连。

mkdir build
cd build
cmake ..
make -j4(j后面的数字表示用多核并行编译，如果你财力雄厚，你甚至可以make -j16，虽然这不一定会更快)
sudo make install
